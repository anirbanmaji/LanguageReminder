# Language Reminder

A simple Android application that displays a floating widget with custom text for each day of the week. This can be used as a reminder for language learning or any other daily task.

## Features

- **Floating Widget**: Displays text on top of other apps.
- **Customizable Mapping**: Set specific text for each day of the week (Monday to Sunday).
- **Persistent Storage**: Saves your mappings so they persist across app restarts.
- **Easy Control**: Start and stop the floating widget directly from the main screen.

## Permissions

- **Overlay Permission**: Required to display the floating widget over other applications.

## How to Use

1. Grant the Overlay Permission when prompted.
2. Enter the desired text for each day of the week in the "Day to text mapping" section.
3. Click "Save Mapping" to store your changes.
4. Click "Start Floating Widget" to show the reminder.
5. Click "Stop Floating Widget" to remove it.
